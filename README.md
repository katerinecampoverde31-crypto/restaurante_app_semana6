# Restaurante App - Semana 6

## Descripción
Este proyecto fue desarrollado para la asignatura de Programación Orientada a Objetos. La aplicación permite gestionar un menú de restaurante utilizando clases, herencia, encapsulación y polimorfismo.

## Funcionalidades
- Agregar platillos y bebidas.
- Mostrar el menú del restaurante.
- Buscar productos por nombre.
- Mostrar información de los productos.

## Estructura del proyecto
```
restaurante_app_semana6/
```
